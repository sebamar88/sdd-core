# Design Skill

## Purpose

Create or refine `design.md`.

## Inputs

- proposal
- delta spec
- repository context

## Outputs

- approach
- decisions
- rejected alternatives
- risks

## Completion

Design is complete when implementation tasks can be planned without inventing major architecture decisions.
