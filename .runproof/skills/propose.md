# Propose Skill

## Purpose

Create or refine `proposal.md` for a change.

## Inputs

- user intent
- exploration notes
- selected profile

## Outputs

- intent
- scope
- non-scope
- risks

## Completion

Proposal is complete when intent and scope can be used by the specifier without chat-only context.
