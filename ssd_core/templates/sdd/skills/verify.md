# Verify Skill

## Purpose

Create or refine `verification.md`.

## Inputs

- requirements
- tasks
- changed files
- project verification mechanisms

## Outputs

- evidence matrix
- commands or checks performed
- manual checks
- known gaps

## Completion

Verification is complete when evidence supports every required behavior or records an explicit gap.
